
# EatWell - The Vegan Food

EatWell is a Frontend Website template of a Vegan Restaurant



## Technology Used

 - ReactJS
 - Tailwind CSS

## Install dependencies
install with npm

```bash
  npm install
```
yarn install
```bash
  yarn install
```

Start the server

```bash
  npm run dev
```
